# Whisper Manager 
Is largely inspired by Wow Instant Messanger, but seeks to expand upon it with added features. 

**Unique Features**
- Global chat history. 
- Recent chats, catch people you missed.
- History search.
- Expanding input box, see your full messages before sending.

**Other Features**
- Flexible data retention configs. (WIP)
- Change UI / Font Sizing

This project is currently in testing, do not rely on it and expect to have your data wiped. As project becomes more stable it will follow...

unstable -> beta > stable
