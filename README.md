# Whisper Manager 
Is largely inspired by Wow Instant Messanger, but seeks to expand upon it with added features. 

**Unique Features**
- Global chat history. 
- Recent chats, catch people you missed.
- Resize text font.
- Resize input (editbox) font.
- Easily search player history.
- Expanding input box, see your full messages before sending.
- Advanced data retention (recent track & long-term track)

**Other Features**
- Change UI / Font Sizing
- Copy chat button.
- Link highlighting
- Emote/OOC highlighting.
- Windows taskbar icon flash (unread message).
- Color messages.
- Notif sound.
- Resize window/postion.
- Change font.
