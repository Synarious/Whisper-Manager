# Whisper Manager 
Is largely inspired by Wow Instant Messanger, but seeks to expand upon it with added features and fixes. It's powerful addon for people interested in whispering people. It stores history far longer than other chat addons, while reducing data bloat to keep your game running fast. 

**Unique Features**
- Global chat history. 
- Recent chats, catch people you missed.
- Emote/OOC highlighting (RP community). 
- Resize text font.
- Resize input (editbox) font.
- Easily search player history.
- Movable windows and buttons, that remember their postion across all your characters. No fuss needed.
- Expanding input box, see your full messages before sending.
- Advanced data retention (recent track & long-term track)

**Other Features**
- Change UI / Font Sizing
- Copy chat button.
- Link highlighting
- Windows taskbar icon flash (unread message).
- Color messages.
- Notif sound.
- Resize window/postion.
- Change font.
- Mispelled support.

**Roadmap/Future**
- Better TRP3 support.
- Themes/window coloring. Art maybe?
- Hardening and bug fixes. 
- Deleting specific user's history.
- Clearing recent chat history manually.
